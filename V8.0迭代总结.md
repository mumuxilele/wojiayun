## 迭代概述

本次V8.0迭代基于产品经理深度评审，针对客户实际使用场景进行功能增强和架构优化，解决V7.0迭代中发现的业务逻辑漏洞和功能缺口。

## 一、产品评审发现的问题

### 评审方法
- 逐行审查三端后端代码（业务逻辑、接口安全、库存管理）
- 对比V7.0待办事项清单
- 分析实际业务场景需求

| 优先级 | 类别 | 具体问题 |
|--------|------|----------|

| P2 | 测试覆盖 | 需增加新功能测试用例 |

## 二、V8.0修复与新增功能

### 2.1 购物车库存校验（P0）

**文件**：`business-userH5/app.py`

修复内容：
- `create_order` 接口增加商品库存校验
- 库存不足时返回明确错误提示
- 订单创建成功后扣减商品库存（`UPDATE business_products SET stock=stock-N`）
- 增加销售数量统计（`sales_count=+N`）

```python
# 库存校验逻辑
product = db.get_one("SELECT stock FROM business_products WHERE id=%s", [product_id])
if current_stock < quantity:
    return jsonify({'success': False, 'msg': f"库存不足（当前库存 {current_stock}）"})
```

### 2.2 评分缓存自动更新（P0）

**文件**：`business-userH5/app.py`

修复内容：
- 用户提交评价后自动计算新的平均评分
- 更新 `business_venues.avg_rating` 和 `business_venues.review_count`
- 更新 `business_shops.avg_rating` 和 `business_shops.review_count`

```python
# 评分缓存更新逻辑
stats = db.get_one("""
    SELECT ROUND(AVG(rating), 1) as avg_rating, COUNT(*) as review_count 
    FROM business_reviews WHERE target_type=%s AND target_id=%s
""", [target_type, target_id])
# 更新venue或shop的评分缓存字段
```

### 2.3 用户配送地址管理（P1）

**新增文件**：
- `business-common/migrate_v8.py` - 数据库迁移
- `business-userH5/app.py` - 新增API

新增API：

| 接口 | 方法 | 功能 |
|------|------|------|
| `/api/user/addresses` | GET | 获取收货地址列表 |
| `/api/user/addresses` | POST | 新增收货地址 |
| `/api/user/addresses/<id>` | PUT | 更新收货地址 |
| `/api/user/addresses/<id>` | DELETE | 删除收货地址 |
| `/api/user/addresses/<id>/default` | POST | 设置默认地址 |

功能特点：
- 支持地址标签（家/公司/其他）
- 支持设为默认地址
- 手机号格式校验
- 数据隔离（按用户ID + ec_id/project_id）

### 2.4 商品规格增强（P1）

**新增文件**：
- `business-common/migrate_v8.py` - 创建SKU表
- `business-admin/app.py` - 管理端商品/SKU管理API

新增表：`business_product_skus`

| 字段 | 说明 |
|------|------|
| product_id | 商品ID |
| sku_name | SKU名称(如:红色-L) |
| specs | 规格JSON |
| price | 价格 |
| stock | 库存 |
| status | 状态 |

新增管理端API：
- `GET/POST /api/admin/products` - 商品CRUD
- `GET/POST /api/admin/products/<id>/skus` - SKU管理

用户端增强：
- 商品详情接口返回 `skus` 列表
- 购物车接口支持 `sku_id` 参数
- 下单时支持规格商品库存校验

### 2.5 评价图片展示增强（P1）

**文件**：`business-userH5/app.py`

增强内容：
