## 一、客户批量导入功能

### 1.1 功能说明
支持通过Excel/CSV文件批量创建客户，自动识别列头、中文转英文

### 1.2 文件位置
- 脚本: `/workspace/create_customers.py`
- 模板: `/workspace/customers_template.csv`

### 1.3 支持的字段

| 接口字段 | 可能匹配的列名 |
|---------|---------------|
| name | 客户名称、姓名、客户、企业名称、公司名称、联系人 |
| phone | 手机号、电话、联系方式、手机 |
| type | 客户类型、类型、客户性质 |
| certType | 证件类型、证件、证书类型 |
| certNo | 证件号码、证件号、身份证号 |
| email | 邮箱、邮件、电子邮箱 |
| province | 省份、省 |
| city | 城市、市 |
| area | 区域、区 |
| address | 地址、详细地址、联系地址 |
| birthday | 生日、出生日期 |
| sex | 性别 |
| nation | 民族 |
| description | 客户描述、备注 |
| category | 客户性质 |
| authCode | 财务编码、客户编码 |

### 1.4 值转换规则

| 字段 | 中文 | 英文 |
|------|------|------|
| 客户类型 | 个人 | P |
| 客户类型 | 企业 | E |
| 客户类型 | 政府 | G |
| 客户类型 | 个体户 | S |
| 证件类型 | 身份证 | 0101 |
| 证件类型 | 营业执照 | 0105 |
| 证件类型 | 护照 | 0104 |
| 证件类型 | 台胞证 | 0111 |
| 性别 | 男 | 0 |
| 性别 | 女 | 1 |

### 1.5 容错处理
- 生日格式: `2001.1.1` → `2001-01-01`
- 企业客户自动使用营业执照
- 个人客户缺少手机号自动生成

## 三、组织架构批量导入

### 3.1 功能说明
支持上传组织架构图片自动识别层级并导入，支持ticket鉴权

### 3.2 文件位置
- 脚本: `/workspace/import_org.py`
- 模板: `/workspace/org_template.csv`

### 3.3 接口信息

| 项目 | 值 |
|------|-----|
| 接口地址 | POST /api/orgUnit/batchSave |
| 鉴权方式 | ticket |

### 3.4 必填字段

| 字段 | 说明 |
|------|------|
| name | 组织名称 |
| number | 组织编码 |
| simpleName | 简码 |
| isCompany | 类型(0区域/1公司/2部门/3备用公司) |
| level | 级别(自动计算) |
| longNumber | 长编码(自动计算) |
| sourceID | 源ID(自动生成) |
| sourceSystem | 来源系统 |
| parentID | 父级ID(自动计算) |
