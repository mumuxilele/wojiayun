📌 接口基础信息

API 网关：https://open.yunzhijia.com/openapi/im/chatbot
鉴权方式：AccessToken + AppKey + AppSecret 签名
请求格式：JSON
字符编码：UTF-8

🔑 鉴权参数

参数名 类型 必填 说明
accessToken string 是 访问令牌
appKey string 是 应用标识
timestamp long 是 时间戳(毫秒)
sign string 是 签名(sha256)

📤 发送消息接口

POST /api/yzj/chatbot/send
Content-Type: application/json

{ "robotId": "bot_xxx", "groupId": "grp_xxx", "msgType": "text", "content": { "text": "消息内容" } }

📥 响应示例

{ "success": true, "code": 200, "msg": "发送成功", "data": { "messageId": "msg_xxx", "sendTime": "2024-01-15 10:30:00" } }
