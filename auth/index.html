接口文档

                

                    
GET /account/customer

                    
客户认证接口，验证签名后返回用户信息

                    
                    
请求参数

                    

                        
参数名类型必需说明

                        
CustomerIdstring是用户唯一标识

                        
Namestring否用户名

                        
Timestampint是当前时间戳（秒）

                        
ExtraInfostring否额外用户信息

                        
Codestring是HMAC-SHA256 签名

                    

                    
                    
签名算法

                    
msg = CustomerId + Name + ExtraInfo + Timestamp
Code = SHA256(HMAC(SECRET_KEY, msg))
