## 核心变更

### 1. 员工端会员管理页面 (P1)

**文件**: `business-staffH5/members.html`

**功能特性**:
- 会员列表展示（头像/姓名/手机号/等级/积分/余额）
- 关键词搜索（姓名/手机号）
- 会员等级筛选（全部/VIP/黄金/白银/青铜/普通）
- 会员详情弹窗
  - 账户信息（余额/积分/等级）
  - 消费统计（累计消费/订单数/签到次数）
  - 积分变动记录

**调用API**: `/api/staff/members`

### 2. 员工端退款管理页面 (P1)

**文件**: `business-staffH5/refunds.html`

**功能特性**:
- 退款统计卡片（待处理/已批准/已拒绝/退款总额）
- 状态筛选（全部/待处理/已批准/已拒绝）
- 退款列表展示（订单号/金额/原因/申请人/时间）
- 批准/拒绝操作（带确认弹窗）
- 详情弹窗（商品信息/操作按钮）

**调用API**: `/api/staff/orders/pending-refunds`

### 3. 员工端首页菜单补充 (P0)

**文件**: `business-staffH5/index.html`

**变更**:
- 新增「退款」菜单入口 (💰)
- 新增「会员」菜单入口 (👥)
- 优化菜单布局为两行

### 4. 退款API增强 (P1)

**文件**: `business-staffH5/app.py`

**增强内容**:
- 支持状态筛选参数 (`?status=pending/approved/rejected`)
- 新增统计数据返回 (`stats: {pending, approved, rejected, total_refund}`)
- 前端状态映射转换

### 5. V12数据库迁移脚本 (P2)

**文件**: `business-common/migrate_v12.py`

**新增字段**:
| 表名 | 字段 | 说明 |
|------|------|------|
| business_members | member_tags | 会员标签，逗号分隔 |
| business_members | source_channel | 来源渠道（h5/app/miniapp/admin）|
| business_members | last_login_at | 最后登录时间 |
| business_orders | refund_operator_id | 退款处理人员工ID |
| business_orders | refund_operator_name | 退款处理人姓名 |
| business_product_skus | low_stock_threshold | 库存预警阈值 |
| business_product_skus | is_low_stock_alert | 低库存预警标志 |
| business_points_log | expired_at | 积分过期时间 |
| business_points_log | source_order_no | 来源订单号 |

**新增视图**:
- `business_low_stock_view` - 库存预警视图
- `business_member_activity_view` - 会员活跃度视图
