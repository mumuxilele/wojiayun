## 五、配置变更提醒

⚠️ **重要**: 由于移除了数据库密码的默认值，部署时必须设置环境变量：

```bash
export DB_PASSWORD=your_production_password
# 或
export MYSQL_PWD=your_production_password
```

## 六、架构改进要点

| 维度 | 改进前 | 改进后 |
|------|--------|--------|
| 密码管理 | 硬编码在3个文件中 | 全部通过环境变量 |
| admin认证 | 存在空pass绕过 | 单一入口统一认证 |
| SQL安全 | 3处字符串拼接 | 全部参数化查询 |
| 并发安全 | 签到无锁 | FOR UPDATE行级锁 |
| 通知可靠性 | 失败即丢失 | 最多3次指数退避重试 |
| XSS防护 | 管理端3处高危 | escHtml统一转义 |
| 部署灵活性 | 绑定固定IP | location.origin动态获取 |
