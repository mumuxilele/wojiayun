#### F3: 真实API环境配置指南 (P0)

```
目标: 提供生产部署配置指南

需配置的API:
1. 微信支付
   - 环境变量: WECHAT_APP_ID, WECHAT_MCH_ID, WECHAT_API_V3_KEY
   - 证书文件: 私钥 + 证书

2. 支付宝
   - 环境变量: ALIPAY_APP_ID, ALIPAY_PRIVATE_KEY, ALIPAY_PUBLIC_KEY
   - 回调URL配置

3. 物流追踪 (快递100)
   - 环境变量: KUAIDI100_KEY, KUAIDI100_CUSTOMER

4. 短信服务 (阿里云)
   - 环境变量: ALIYUN_SMS_ACCESS_KEY, ALIYUN_SMS_TEMPLATE

交付物:
- docs/API_CONFIG_GUIDE.md (配置指南)
- .env.example (环境变量模板)
```
