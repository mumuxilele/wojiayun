#### 问题 7：评价体系缺少积分奖励闭环

**现状**：
- 用户可以评价商品，但没有积分奖励
- 商家回复后没有通知用户
- 差评没有主动安抚机制

**优化方案**：
```python
# 评价后触发积分奖励
def reward_review(user_id, review_id, rating):
    rewards = {
        5: 20,   # 5星好评：20积分
        4: 10,   # 4星：10积分
        3: 5,    # 3星：5积分
        1: 0,    # 1-2星差评：0积分
        2: 0,
    }
    points = rewards.get(rating, 0)
    if points > 0:
        MemberService.add_points_log(
            user_id=user_id,
            points=points,
            change_type='review_reward',
            change_reason=f'评价获得{rating}星奖励'
        )
```
