## 三、服务详细说明

### 3.1 发票服务 (InvoiceService)

#### 核心功能
- **发票抬头管理**：个人/企业抬头创建、修改、删除、默认设置
- **发票申请**：从订单申请发票、支持电子发票/纸质发票
- **发票审核**：审核通过/驳回
- **发票开具**：开具发票、生成PDF地址
- **发票作废**：支持作废和红冲
- **发票统计**：按状态/类型/抬头类型统计

#### API接口
```
用户端:
  GET  /api/user/invoice/titles          # 获取抬头列表
  POST /api/user/invoice/titles          # 创建抬头
  PUT  /api/user/invoice/titles/<id>     # 修改抬头
  DELETE /api/user/invoice/titles/<id>  # 删除抬头
  GET  /api/user/invoices                # 获取发票列表
  GET  /api/user/invoices/<id>          # 发票详情
  POST /api/user/orders/<id>/invoice/apply  # 申请发票
  POST /api/user/invoices/<id>/cancel    # 取消发票

管理端:
  GET  /api/admin/invoices               # 发票列表
  GET  /api/admin/invoices/<id>          # 发票详情
  POST /api/admin/invoices/<id>/approve  # 审核通过
  POST /api/admin/invoices/<id>/reject   # 驳回
  POST /api/admin/invoices/<id>/issue    # 开具发票
  POST /api/admin/invoices/<id>/cancel   # 作废发票
  GET  /api/admin/invoices/statistics     # 发票统计
```

### 3.2 备份服务 (BackupService)

#### 核心功能
- **全量备份**：mysqldump 完整备份
- **增量备份**：备份数据变更
- **压缩存储**：支持 gzip 压缩
- **备份恢复**：完整恢复/表级恢复
- **备份验证**：文件完整性校验（MD5）
- **自动清理**：保留最近30天备份

#### 配置参数
```python
BACKUP_DIR = '/var/backup/mysql'     # 备份目录
MAX_BACKUP_RETENTION = 30              # 保留天数
ENCRYPTION_KEY = os.environ.get(...) # 加密密钥（可选）
```

### 3.3 任务队列服务 (TaskQueueService)

#### 核心功能
- **任务类型**：通知/批量订单/统计分析/积分结算/库存同步/数据导出/数据清理
- **优先级调度**：支持LOW/NORMAL/HIGH/URGENT四级优先级
- **定时执行**：支持定时任务调度
- **重试机制**：自动重试，失败次数可配置
- **状态管理**：pending/running/completed/failed/cancelled/retrying

#### 任务处理器注册
```python
from business_common.task_queue_service import task_queue_service

def my_handler(task_data):
    # 处理任务逻辑
    return {'success': True, 'result': task_data}

task_queue_service.register_handler('custom_task', my_handler)
```

### 7.2 备份恢复流程

```python
from business_common.backup_service import backup_service

# 创建备份
result = backup_service.create_backup(
    backup_type='full',
    compress=True,
    description='手动备份'
)

# 验证备份
verify_result = backup_service.verify_backup(backup_id)

# 恢复备份
restore_result = backup_service.restore_backup(
    backup_id='BK20260405XXXXXXXX',
    force=False
)
```

### 7.3 任务队列使用

```python
from business_common.task_queue_service import task_queue_service

# 注册处理器
def batch_notification_handler(data):
    user_ids = data.get('user_ids', [])
    message = data.get('message', '')
    # 批量发送通知逻辑
    return {'success': True, 'sent': len(user_ids)}

task_queue_service.register_handler('notification', batch_notification_handler)

# 创建任务
result = task_queue_service.create_task(
    task_type='notification',
    task_data={'user_ids': [1,2,3], 'message': '系统通知'},
    priority=10,
    ec_id=1
)
```
